# Virtual Omise Town

## setup
```
python3 -m venv ~/venv
source ~/venv/bin/activate
pip install -r requirements.txt
```

## usage
```
flask run -h localhost -p 5001 \
--reload --extra-files templates/index.html \
--debugger
```
